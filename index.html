sys9wm

A lightweight and familiar concept window manager inspired by Classic Mac OS.

  • Low-overhead for low-powered systems
  • Written in low-level C for best performance
  • POSIX-compatible, making it portable to any *NIX system
  • Modular and extendable
  • Easily configurable and modifiable

Tweeter

Mock Twitter embed bot for Discord.

  • User information pasted into tweet embed (with support for nicknames)
  • Randomized likes and retweets
  • 100 percent accurate recreation of the real Discord embed
  • All commands and usage logged to console output.
  • Lightweight (less than 100 lines)
